# CityFix

1. Clone repo `git clone https://github.com/ardit203/CityFix.git`
2. Switch branch to dev `git switch dev`
3. Open `Docker-Desktop`
4. Execute `docker-compose up -d` in the terminal
5. Open backend with `IntelliJ` and run `BackendApplication.java`
6. Open frontend with `IntelliJ / WebStorm` and first execute `npm install` and then `npm start dev` in the terminal.
7. You can open `swagger-ui` in the endpoint `http://localhost:8080/swagger-ui/index.html#/`
